Feedstock for [covfie](https://github.com/acts-project/covfie) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Library for storing interpolatable vector fields on co-processors

## Install

```bash
conda install -c hep-forge -c conda-forge covfie
```
